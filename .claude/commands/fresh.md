# Fresh Start Prep

End this session and prepare a clean handoff to a new one.

## First, do everything in /save-progress
1. Review what changed this session (`git log`, `git diff`).
2. Update MEMORY.md — current status, decisions, gotchas, preferences.
3. Update `LESSONS.md` with any new mistakes-turned-rules.
4. Add learning notes to `docs/learning/` for concepts explained.

## Then prepare the handoff
5. If there's uncommitted work, ask whether to commit before closing.
6. Write a "Next Session" note at the top of MEMORY.md:
   - What task was in progress
   - The next concrete step
   - Any pending decisions
7. Show a brief summary: what was saved, what to pick up next, and suggest typing
   `/start-session` in the new terminal.
