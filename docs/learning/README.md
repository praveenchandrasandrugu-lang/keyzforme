# Learning Notes

Plain-English notes on concepts you learn while building Keyzforme. Grown via the `/learn`
command (or just by asking). One file per topic, e.g.:

- `nextjs.md` — App Router, Server vs Client Components, routing, data fetching
- `typescript.md` — types, strict mode, generics, common errors
- `css-tailwind.md` — Tailwind utilities, responsive design, layout
- `git.md` — branches, commits, PRs, undoing things
- `deployment.md` — Vercel, environment variables, preview deploys
- `web-basics.md` — HTTP, the browser, rendering

Each entry: a one-line plain explanation + a concrete example from our code + the date.
This is for *you* — write it the way you'd want to read it in three months.
